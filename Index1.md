Palindrome Checker

This program checks for if a word or phrase is a palindrome. A palindrome reads the same forwards and backwards regardless of puncuation, case, and spaces. 

To use this program:
1.Enter a word or phrase when told to.
2. The program will remove all punctuation and spaces from what you input and decide if it is a plaindrome.
3. Output will inform you whether the input is a palindrome or not. 

Example:
  Enter a word or phrase: Racecar
  RACECAR is a palindrome.
